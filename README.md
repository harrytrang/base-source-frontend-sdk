# Base Source Frontend Ssdk Package

---

## Prerequisites

- **Node.js** >= 18.x
- **npm** >= 9.x

---

## Getting Started

### 1. Build the package

```bash
npm run build
```

### 2. Use the package in your project

```bash
npm install base-source-frontend-sdk
```

```javascript
import { createClient } from "base-source-frontend-sdk";
```
